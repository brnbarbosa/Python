What is the mental model for csv.DictReader — what does it take as input and what does it produce?	It wraps an already-open file object (not a path). It yields each data row as a dict with column headers as keys.	python::csv
What is the accumulator pattern for grouping data into a nested dict?	setdefault the outer key to {}, then setdefault the inner key to 0.0, then += the value. Never seed setdefault with the value itself — always seed with the zero.	python::patterns
When should you use dict.setdefault() vs plain assignment (=)?	setdefault when you want to preserve an existing value (initialize-only). Plain = when you always want to overwrite. Totals always use =.	python::patterns
What does str.join() take as its argument, and what is the most common mistake?	It takes an iterable of strings. Common mistake: calling it inside a loop on one string instead of once on the whole list. "\n".join(lines) — one call, outside the loop.	python::builtins
What is the rule for data functions vs display functions?	Data functions return data. Display functions print. Never mix the two — a function that reads a CSV must return the data, not print it.	python::structure
Why does open_csv_file() use enumerate() and what is the idx + 2 offset for?	enumerate() tracks the loop index so we can report which row is bad. idx starts at 0 and skips the header row, so the human-readable row number is idx + 2.	python::builtins
What is the correct way to get the parent directory of the current file in pathlib?	Path(__file__).resolve().parent — use .parent for one level up, .parents[n] only when climbing multiple levels.	python::pathlib
What exception should you catch when a CSV file is missing, and what's the right response?	FileNotFoundError — catch it and call sys.exit() with a clear, actionable message. Never let the raw traceback reach the user.	python::error-handling
What two keyword arguments make Path.mkdir() safe to call unconditionally?	parents=True (creates missing parent directories) and exist_ok=True (silently succeeds if directory already exists). Without them, mkdir() crashes if the path exists or has missing parents.	python::pathlib
What is the difference between datetime.strptime() and datetime.strftime()?	strptime = "parse time" — converts a string INTO a datetime. strftime = "format time" — converts a datetime INTO a string. You have a datetime and need a string → strftime.	python::datetime
In automated scripts, why use logging instead of print()?	print() output goes nowhere when no terminal is open. logging writes to a file, so you can inspect what happened after the fact. logging also adds timestamps and severity levels automatically.	python::logging
When is catching `except Exception as e` acceptable vs a bare `except:`?	Use `except Exception as e` in long-running automated processes where you want to log any failure and keep running. Never use bare `except:` — it catches KeyboardInterrupt and SystemExit, blocking the user from stopping the script.	python::error-handling
What does requests.get() return, and what are the two most important attributes on it?	A Response object. response.status_code (200 = OK, 4xx = client error, 5xx = server error) and response.json() (parses the body into a Python dict).	python::requests
How do you pass query parameters cleanly with requests.get()?	Use the params= dict argument — requests builds the query string automatically. Never hardcode params in the URL string.	python::requests
What is the right place to call load_dotenv() and why?	At module level, just below imports — runs once when the module is imported. Never inside a function, which would re-run it on every call.	python::patterns
What is the correct boundary check when validating a user-selected index against a list?	`pick < 0 or pick >= len(results)` — valid indexes run 0 to len-1. Using `> len(results)` lets len itself through, which is one past the end.	python::patterns
How do you safely loop until a user enters valid input in Python?	`while True:` with a `try/except` block — attempt conversion/validation, `break` on success, `except` catches bad input and prints an error to retry.	python::patterns
When you enumerate a list, what does each iteration yield?	A tuple of `(index, item)` — `for idx, item in enumerate(my_list)`. Use it whenever you need both the position and the value inside a loop.	python::builtins
Before writing a return type annotation, what should you verify?	Look at the actual `return` statement — what type does it produce? The annotation must match reality, not intent.	python::structure
What is JSON Lines (.jsonl) and why use it instead of JSON for append-heavy storage?	One independent JSON object per line, no wrapping array. To append: open in "a" mode, write one line with json.dumps() + "\n". Regular JSON requires reading, parsing, and rewriting the whole file.	python::patterns
How do you convert a datetime object to a readable string for storage?	Call .isoformat() on it — produces a standard string like "2026-04-06T15:55:13.424037". datetime objects cannot be JSON-serialized directly.	python::datetime
What is the rule for adding try/except blocks?	Only catch exceptions that can actually be raised. Before adding a try/except, ask: under what real conditions does this exception fire? Never catch the impossible.	python::error-handling
How do you write one JSON record to a .jsonl file in append mode?	open(filename, "a", encoding="utf-8") as f: then f.write(json.dumps(record) + "\n") — the \n is required so each record lands on its own line.	python::patterns
What is a pandas DataFrame and what everyday tool is it equivalent to?	A DataFrame is a table of rows and columns in memory — equivalent to a spreadsheet tab. You can filter, sort, group, and aggregate it in one or two lines instead of writing loops.	python::pandas
How do you load a CSV file into a pandas DataFrame?	pd.read_csv(path) — pass a Path or string directly. Never wrap it in with open(). pandas manages the file lifecycle internally.	python::pandas
What is the universal import alias for pandas and why does it matter?	import pandas as pd — every tutorial, colleague, and Stack Overflow answer uses pd. Deviating from it makes your code harder to read for anyone else.	python::pandas
How do you filter rows in a DataFrame where a column equals a specific value?	df[df['column'] == 'value'] — boolean indexing: the inner expression produces a Series of True/False, the outer df[] keeps only the True rows.	python::pandas
What is the pandas pattern for grouping rows by a column and summing another column?	df.groupby('group_col')['value_col'].sum() — returns a Series where the index is unique group values and values are the sums.	python::pandas
How do you apply string methods (like .lower() or .strip()) to a pandas Series?	Use the .str accessor: series.str.lower(), series.str.strip(). Never call string methods directly on a Series — they don't exist there.	python::pandas
How do you combine multiple Series into one labeled structure for CSV export?	pd.concat([s1, s2, s3], keys=['Label 1', 'Label 2', 'Label 3']) — creates a MultiIndex Series with each section labeled. Chain .to_csv(path) to export.	python::pandas
What is the difference between a pandas Series index and its values?	series.index gives the labels (row names). series.values gives the raw data array. For a groupby result, index = group names, values = aggregated numbers.	python::pandas
How do you convert a DataFrame date column from string to datetime?	df["col"] = pd.to_datetime(df["col"]) — assign the result back. Do this in the loader so all downstream code gets proper datetime objects.	python::pandas
How do you calculate the number of days between today and each date in a pandas datetime Series?	(pd.Timestamp.today() - df["date"]).dt.days — subtract a Timestamp from the Series, then use .dt.days to extract integer days element-wise.	python::pandas
How do you select multiple columns from a DataFrame?	df[["col1", "col2", "col3"]] — double brackets. Outer [] is the indexing operator; inner [] is the list of column names. Single bracket returns a Series; double brackets return a DataFrame.	python::pandas
What is the .dt accessor in pandas and when do you use it?	.dt exposes datetime and timedelta attributes element-wise on a Series — e.g. .dt.days, .dt.year, .dt.month. Use it like .str, but for date/time operations instead of string operations.	python::pandas
What is the rule for directory creation relative to file writes?	Always call mkdir() before any file write. Directory creation is setup — it belongs at the top of the function, before any I/O. Writing to a path whose parent doesn't exist raises FileNotFoundError.	python::pathlib
What does index=False do in DataFrame.to_csv() and when should you use it?	Suppresses the DataFrame's integer row index from being written as a column. Use it whenever the index has no meaning (i.e., it's just 0, 1, 2...) — otherwise you get an unnamed first column in your CSV.	python::pandas
What is the pipeline order in a well-structured main() that loads data, analyzes it, and exports results?	1) Create output directories (mkdir). 2) Load data. 3) Run analysis. 4) Write reports/exports. Setup always before I/O — never interleave them.	python::structure
What are the three steps of a pygame game loop, in order?	1) Handle input/events. 2) Update (move objects, apply physics, check collisions). 3) Draw. Drawing last ensures you render the current frame's state, not last frame's.	python::pygame
Why does pygame need pygame.display.flip() after drawing?	pygame draws onto an invisible back buffer. flip() swaps it to the visible screen in one instant, preventing flickering. Without it, nothing you draw is ever shown.	python::pygame
What does pygame.time.Clock() do and where should clock.tick(fps) be called?	Clock manages frame timing. tick(fps) pauses until 1/fps seconds have elapsed, capping the frame rate. Call it at the very end of the loop, after all input/update/draw work is done.	python::pygame
What does pygame.Rect(x, y, width, height) represent and what does the y-axis direction mean?	A rectangle defined by top-left corner (x, y) and dimensions. In pygame, y=0 is the top of the screen and y increases downward — so y=HEIGHT-rect_height places the object at the bottom.	python::pygame
How do you simulate gravity in a pygame game loop?	Every frame: vertical_velocity += GRAVITY, then player.y += vertical_velocity. Gravity is a constant added to velocity each frame — never added directly to position.	python::pygame
Why use elif player_vertical_velocity >= 0 for floor clamping instead of else?	A plain else resets velocity even when the player just jumped (negative velocity), cancelling the jump immediately. elif >= 0 only clamps when falling or stationary — lets negative (upward) velocity pass through.	python::pygame
What are the two things floor clamping must do when a player lands?	1) Reset vertical_velocity to 0 so gravity stops accumulating. 2) Set player.bottom = HEIGHT to snap the rect to the exact floor line, preventing pixel-level drift below the surface.	python::pygame
What is the naming convention for pygame key constants and why does case matter?	pygame.K_SPACE, pygame.K_LEFT, pygame.K_UP — always K_ prefix, all uppercase. Python is case-sensitive: pygame.k_space does not exist and raises AttributeError.	python::pygame
How do you fire a custom event on a repeating timer in pygame?	pygame.time.set_timer(event_id, milliseconds) — call it once before the loop. Use pygame.USEREVENT + 1 as the event_id. Catch it in the event loop with if event.type == event_id.	python::pygame
Why must you iterate over obstacles[:] instead of obstacles when removing items mid-loop?	Modifying a list while iterating over it shifts indices and causes items to be skipped. obstacles[:] creates a shallow copy — you iterate the copy, remove from the original.	python::patterns
What is the correct draw order in a pygame frame to make text appear on top of everything?	1) window.fill() — clear the screen. 2) Draw game objects. 3) Blit text/UI. 4) pygame.display.flip(). Anything blitted after flip() is invisible until the next frame.	python::pygame
How does pygame.Rect.colliderect() work and what does it return?	player.colliderect(other_rect) returns True if the two rectangles overlap, False otherwise. Check it every frame in the update section — not in the event loop.	python::pygame
What is __init__ and when does Python call it?	__init__ is the initializer — Python calls it automatically when you create an instance with ClassName(). Set up all attributes here with self. Never call __init__ directly.	python::oop
What is the difference between a local variable and a self attribute inside __init__?	Local variable (x = 1) dies when __init__ returns. Self attribute (self.x = 1) lives on the object and is accessible from all methods. Always use self. for anything the object needs to remember.	python::oop
What does encapsulation mean in practice?	An object owns and manages its own data. Instead of main.py doing player.rect.topleft = (10,380), expose player.reset() — the caller says what to do, not how. Never reach into another object's internals.	python::oop
When should logic live in a class method vs. in main.py?	Logic that only needs data the class owns belongs in the class. Logic that requires data from multiple objects (e.g. collision between player and obstacle) belongs in the orchestrator (main.py).	python::oop
What is a Flask Blueprint and why use it?	A Blueprint groups related routes into a separate file. Register it in app.py with app.register_blueprint(bp). Keeps routes organized by feature instead of cramming everything into app.py.	python::flask
What is the difference between GET and POST in HTTP?	GET retrieves data — used when a browser requests a page. POST sends data to the server — used when a form is submitted. A Flask route handles both with methods=["GET", "POST"].	python::flask
How do you check which HTTP method was used inside a Flask route?	request.method returns "GET" or "POST" as a string. Use if request.method == "POST": to branch between showing a form and processing it.	python::flask
What are the four steps to execute a SQL statement with sqlite3?	1) conn = sqlite3.connect("file.db"). 2) cursor = conn.cursor(). 3) cursor.execute("SQL..."). 4) conn.commit() then conn.close(). Commit saves changes — without it, writes are lost.	python::sqlite
What is the correct SQL syntax for a foreign key in SQLite?	Declare the column normally (transaction_id INTEGER), then add a separate constraint line at the end: FOREIGN KEY (transaction_id) REFERENCES other_table(id). Never inline it with the column type.	python::sqlite
What does @app.route("/") do in Flask?	It's a decorator that maps a URL path to the function below it. When a browser visits that URL, Flask calls the function and sends its return value as the HTTP response.	python::flask
Why store calculated values (like discount) in the database instead of always computing them?	When the inputs can change later (e.g. due date extension), a stored value stays frozen at the original calculation — preserving the historical truth. Recomputing would silently change the record.	python::patterns
How do you make a Jinja2 template inherit from a base layout?	In the child: {% extends "base.html" %} at the top, then {% block content %}...{% endblock %} around the unique content. In base.html: {% block content %}{% endblock %} marks where child content is injected.	python::flask
How do you loop over a list in a Jinja2 template?	{% for item in items %} ... {{ item[0] }} ... {% endfor %} — same as Python but with {% %} for logic and {{ }} for output. One <tr> per iteration renders one table row per item.	python::flask
What is the correct way to add days to a Python date object?	from datetime import timedelta — then date + timedelta(days=n). To generate installment n: first_date + timedelta(days=n * interval). Never add integers directly to a date.	python::datetime
